# Changesets

This directory contains implementation changesets for the agent.

Each changeset maps to one GitHub issue and keeps the working task list for that issue.

## Naming

Use one file per issue with this pattern:

`<issue-number>-<short-kebab-title>.md`

## Contents

Each changeset file contains:

* the GitHub issue number and title
* the goal of the changeset
* the scope and non-goals
* the relevant design references
* the ordered implementation task list
* the verification tasks derived from the quality requirements
